Html

(ing. Hypertext Markup Language) — brauzerin oxuya biləcəyi hər hansi sənədi və ya səhifəni yaratmaq üçün xüsusi hipermətn dilidir. HTML internetin fundamental baza texnologiyasidir və veb-səhifənin növünü, funksiyasini təyin edən diskriptorlar əsasinda yaradilmiş dildir. Bütün veb-qovşaqlarda toplanmiş və internetə qoşulmuş kompüterlərin ekranlarinda göstərilən sənədlər HTML proqram kodunda yazilirlar. HTML - səhifələrdə mətn bloklarinin, təsvirlərin yerləşdirilməsinə, cədvəllərin qurulmasina, sənədin və sənəddəki mətnin rənglərinin seçilməsinə, multimediya elementlərinin əlavə edilməsinə, hiperistinadlarin və bütün bu elementlər arasinda əlaqələrin yaradilmasina imkan verir. Hipermətn koduna malik olan fayllar (.htm) və (.html) genişləndirməsi ilə göstərilir.

CSS

(ing. Cascading Style Sheets) — sənədin xarici görünüşünün, işarələmə dilinin istifadəsi ilə yazilmiş, formal təsvir dili. Adətən HTML və XHTML işarələmə dillərində yazilmiş veb-səhifələrin təsviri və tərtibati üçün istifadə edilir, lakin istənilən digər XML sənədlərə, məsələn, SVG və ya XUL-a da tətbiq edilə bilər. HTML sənədlərini yaradarkən veb-dizaynerlər müxtəlif mürəkkəb məsələlərlə qarşilaşilar: müəyyən abzas daxilində şifrələrin dəyişdirilməsi; mətnin və ya təsvirin sənədin hər hansi bir yerində yerləşdirilməsi; mətnin və ya fonun rənginin dəyişdirilməsi; müxtəlif verilənlər cədvəllərinin format edilməsi və s. Bu işləri standart üsulla da yerinə yetirmək olar, yəni HTML dilində hər bir abzasi ayrica əmrlər yiğimi ilə təsvir etmək mümkündür. Bu halda proqramin yazilmasi prosesi həddən artiq çətinləşəcək, sənədin ölçüsü isə çox böyük olacaq. Bu işi sadələşdirmək üçün səhifələrə CSS standartinda yerinə yetirilmiş xarici fayl qoşulur və həmin faylda istifadə olunmuş xüsusi makrodil vasitəsilə səhifə birdəfəlik formatlaşdirilir. Başqa sözlə, CSS fayli burada şablon kimi istifadə edilir ki, onun köməyilə HTML sənədlərində mətnlər, cədvəllər və digər elementlər formatlaşdirilir. CSS standartindan heç bir məhdudiyyət qoyulmadan istənilən serverdə istifadə etmək olar. CSS kodlari, HTML kodlarinin içinə yazila bildiyi kimi xarici bir CSS fayli yaradilaraq da əməliyyat yaradila bilir. Növünə görə ya bölmələrində ala bilərlər. Bundan başqa,koduyla CSS fayli iş səhifəsi əlavə oluna bilər. Onlari ayri-faylda və ya fayllarda saxlamaq, onlari dəyişik HTML səhifələrində istifadə edilməsini təmin edir, eyni səhifənin dəyişik cihazlara görə formatlandirilmasini asanlaşdirir. CSS kodlarinin HTML faylina əlavə olunmasi üç fərqli şəkildə icra edilə bilər. CSS standartinda fayllarin formati .css şəklindədir.
